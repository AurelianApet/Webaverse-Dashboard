Webaverse Guide

At last. Welcome!

The following is the earliest version of a guide to help you equip yourself with a series of tools in order to participate in the open metaverse on the web, or Webaverse.

Depending on what you would like to do, create, and own. You will need the following tools:

  1. Spatial Web Browser
  2. Real Time Communications
  3. Spatial Creation Tool
  4. Spatial Package Manager & Directory

1. Spatial Web Browser

Metassium

A special version of the Chromium web browser, that is configured to allow you to run any existing WebXR application as an AR app inside of VR.

To get started download the Metassium browser from Here

Extract chrome.7z from the 7zip file you have downloaded.

Extract the Chrome-Bin folder out of chrome.7z making sure to have the folder "Chrome-bin" exist on a directory on your machine with no spaces in the path.

On a computer, running SteamVR run "metachromium.cmd" located in the Chrome-bin direcotry

You should see a spinning blue cube in the browser window. In order to render to the headset as an AR overlay, click "EnterXR"


This has been tested with many WebXR apps and seems to work with all of them. You can navigate a single tab of Metassium and click the apps' "EnterXR" button and have it render alongside any SteamVR content

2. Real-Time Communicator

MetaRTC

Meta RTC is a tool that allows you to connect to a room and the people in it in real-time given a context you choose to join using a Spatial Web Browser.

Enter a room code on your wrist, or navigate in from a URL, and instantly be able to connect with others with that code with real time metadata, voice, and video all over the WebRTC standard.

3. Spatial Creation Tool

Cryptopolys

Not only should you be able to experience spatial web objects or AR apps. You should be able to create them in the spatial web. Cryptopolys allows you to create and author unique spatial objects, visually that you own using a web browser.

4. Spatial Package Manager

XRPackage

XRPackage is a package manager for the open web metaverse. It is also the default package manager for the open web metaverse.

  • Browse
    • XRPackage is a destination to browse other people’s creations
  • Download & Install with npm 
    • npm install -g xrpk
  • Or use the Create page to upload a file to the network.
